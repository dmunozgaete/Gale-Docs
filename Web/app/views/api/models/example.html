Un archivo DBML , es un archivo que define un esquema o agrupación de modelos de la base de datos conectada (SQL Server, Oracle, MySql, MongoDB , etc.).

En GALE, el contenedor del esquema usado para la API, es el archivo dbml, y este puede contener modelos conectados con la base de datos o no.

Estos modelos son vinculados a través de los servicios creados para tu proyecto o creando un endpoint (Controlador API) heredando de la clase base BlueprintController que generará de forma automatica el principio RESTFul.

Creación y Exposición de un Modelo con BlueprintController.

Uso Basico
